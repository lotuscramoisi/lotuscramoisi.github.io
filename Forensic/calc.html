Using file on calc.exe: PE32 executable (GUI) Intel 80386, for MS Windows, UPX compressed, 3 sections We can see that it is a Windows executable that was compressed using UPX. With that knowledge, we can just unpack using the following command. 
upx -d calc.exe -o unpackcalc.exe
Once unpcaked, I used ghidra but saw nothing at first glance. I used the string command on the executable on can see this weird html code.
